# Pinyin Practice

Pinyin Practice 是一个面向低龄中文学习者的拼音课后练习网站，适合幼儿园到小学一年级学生在课后进行声母、韵母、整体认读音节和拼读复习。

网站采用卡片式练习界面，尽量减少无关操作，让学生可以直接看拼音、听读音、反复练习。项目不包含登录、注册、广告、后端服务、数据库或学生信息收集。

## 在线使用

访问 GitHub Pages：

https://splashcurry.github.io/pinyin-practice/

打开网站后，可以从首页进入不同练习模块。建议按照课堂学习顺序练习，也可以直接使用随机练习进行巩固。

## 功能简介

- 单个拼音练习：复习声母、韵母和整体认读音节。
- 四声练习：选择韵母后练习一声、二声、三声、四声。
- 拼读练习：练习“声母 + 韵母 + 声调”的组合读法。
- 课程练习：按课次组织练习内容，适合课后跟进复习。
- 每日练习：按日期查看当天安排的练习内容。
- 历史练习：查看已经配置过的每日练习。
- 随机练习：从拼读内容中随机抽题，适合反复巩固。

## 如何使用

1. 打开在线网站：https://splashcurry.github.io/pinyin-practice/
2. 在首页选择练习类型。
3. 点击卡片上的播放按钮听读音。
4. 使用“上一个”“下一个”或页面中的选项切换练习内容。
5. 需要复习当天内容时，进入每日练习或课程练习页面。

## 适用场景

- 学生课后自主练习拼音。
- 家长陪同孩子进行每日复习。
- 教师给学生布置简单的课后拼读练习。
- 作为开源静态网站示例，用于展示 React + Vite 的轻量教育应用。
